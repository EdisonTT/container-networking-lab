# 📖 Network & System Abbreviations

| Short | Expanded Form | Description |
| --- | --- | --- |
| **ARP** | Address Resolution Protocol | Maps an IP address to a physical MAC address on a local network. |
| **MAC** | Media Access Control | A unique hardware address assigned to a network interface. |
| **IPAM** | IP Address Management | Docker's built-in system for allocating IP addresses to containers. |
| **DNAT** | Destination NAT | Modifies the destination IP/port of an incoming packet (e.g., port forwarding). |
| **SNAT** | Source NAT | Modifies the source IP of an outgoing packet so replies can find their way back. |
| **NAT** | Network Address Translation | The general process of modifying IP addresses in packet headers while in transit. |
| **VETH** | Virtual Ethernet | A virtual network cable with two ends used to connect network namespaces. |
| **NIC** | Network Interface Card | The physical (or virtual) hardware component that connects a device to a network. |
| **PID** | Process ID | A unique number assigned by the Linux kernel to an active process. |
